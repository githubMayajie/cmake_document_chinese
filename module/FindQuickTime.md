QUICKTIME_FOUND, if false, do not try to link to gdal
QUICKTIME_INCLUDE_DIR, where to find the headers  

$QUICKTIME_DIR is an environment variable that would correspond to the
./configure ¨Cprefix=$QUICKTIME_DIR
